Streamlit To-Do List Web App



Project Overview
- This project is a simple interactive To-Do List web application built using Streamlit and Python. It allows users to add and display tasks in a browser-based interface.



Objective
- To build a lightweight web application that demonstrates basic frontend interaction using Python and Streamlit.



Tools Used
- Python
- Streamlit
- VS Code / Jupyter Notebook



Features
- Add tasks using input box
- Display list of tasks dynamically
- Simple interactive UI
- Browser-based execution using Streamlit



How It Works
- User enters a task in the input field
- Clicks “Add Task” button
- Task is stored in a list
- All tasks are displayed below in real-time



How to Run the App

1. Install Streamlit:

pip install streamlit


2. Run the app:

streamlit run todo_app.py


3. Open browser link provided in terminal
